# VR Alpha Sim - Drone Control

## VR Drone Controls
- **Left Controller Thumbstick**:
  - **Up/Down**: Thrust (lift up/down)
  - **Left/Right**: Forward/backward movement
- **Right Controller Thumbstick**:
  - **Left/Right**: Yaw (turn left/right)
  - **Up/Down**: Pitch (tilt forward/backward)

## Features
- VR controller-based physics simulation
- Green drone box positioned at (0, 2, 0) in the scene
- Automatic stabilization when thumbsticks are centered
- Can interact with other physics objects in the VR environment

## Running the Simulation
Open the project in Godot and run the Main.tscn scene. Use your VR controllers to fly the drone around the environment.
